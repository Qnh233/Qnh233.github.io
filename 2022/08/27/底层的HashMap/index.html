底层源码HashMap

可以看到Set集合以及Map集合，其底层都是使用了HashMap。数组+链表+红黑树

区别是Set的Value值是设定好的一个默认值，而Map中Value是可以设定的。

相同点

  1. 使用几乎相同的源代码
  2. 扩容机制相同
  3. Hash的求值都是利用Key变量
  4. 排除特定的子类，元素都是无序且不可重复的（LinkedHashSet是有序的）

区别

Set Map
Value为默认常量值 Value可以更改
无K-V 有K-V
Set的Key就是值 具有KeySet，Values，EntrySet方法
